## Development

```
npm install

npm start
```

#### subscribe ethereum real-time events

this project uses Infura Ethereum API.  
you need to get ProjectID and set `.env` file as below.

```
REACT_APP_INFURA_PROJECT_ID="YOUR_INFURA_PROJECT_ID"
```

## Build

```
npm run build
```
